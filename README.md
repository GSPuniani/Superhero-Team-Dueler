# Superhero-Team-Dueler
<i>Make School CS 1.1: Homework #2</i>

This project is a Python game that can be played from the Command Line by running `arena.py` in the Terminal. The user is prompted to build two teams of heroes, and the game runs until one team wins (or the teams tie). This project was made with the following tutorial: https://www.makeschool.com/academy/track/superhero-team-dueler
